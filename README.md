# 🏏 IPL Analytics Dashboard

A ball-by-ball IPL cricket analytics web application built with Python and Streamlit.

## 🔍 Features
- Processes 500K+ ball-by-ball deliveries across IPL seasons
- Computes real-time win probabilities
- Interactive dashboard with player & team statistics
- Data sourced from Cricsheet (public cricket data repository)

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib / Plotly
- JSON data processing

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
